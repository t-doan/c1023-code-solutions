# node-intro-quiz-notes

## Quiz Questions

After completing this exercise, you should be able to discuss or answer the following questions:

- What are the 3 components of a fullstack Web architecture?
  Front-end, Back-end, Database

- What is Node.js?
  Allows javaScript to run outside of a web browser

- What can Node.js be used for?
  Script, automate, write in JavaScript

- What is a REPL?
  Read-Eval-Print-Loop simple interactive environment that takes a single input and execute then return

- When was Node.js created?
  2009

- What backend languages have you heard of?
  Ruby, Java, Python, JavaScript, C#, PHP, Golang
